# PS5 Vault 🎮

Mobilny tracker gier PS5 — dark theme, RAWG API, localStorage.

**[▶ Otwórz aplikację](https://matiseekk-dot.github.io/ps5-vault/)**

## Stack
- React 18 + Vite
- Recharts
- RAWG.io API

## Lokalne uruchomienie
```bash
npm install
npm run dev
```

## Deploy
Push na `main` → GitHub Actions buduje i wgrywa na GitHub Pages automatycznie.
