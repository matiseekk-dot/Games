PS5 Vault

✓ Nie zbieramy Twoich danych
PL · English

1. Informacje ogólne

PS5 Vault ("Aplikacja") to osobisty tracker gier na PlayStation 5. Aplikacja jest dostępna bezpłatnie i nie wymaga rejestracji ani logowania. Twórca aplikacji: SkuDev (Polska).

2. Jakie dane przechowujemy

Wszystkie dane (lista gier, statusy, oceny, notatki, dane finansowe, sesje grania, budżet) są przechowywane wyłącznie lokalnie na Twoim urządzeniu w pamięci przeglądarki (localStorage). Dane te nigdy nie są przesyłane na żadne serwery zewnętrzne.

3. Dane które NIE są zbierane

Aplikacja nie zbiera, nie przechowuje ani nie przekazuje żadnych danych osobowych, w tym: imienia i nazwiska, adresu e-mail, numeru telefonu, lokalizacji, danych finansowych, identyfikatorów urządzenia ani identyfikatorów reklamowych. Aplikacja nie używa Google Analytics, Facebook Pixel ani żadnych innych identyfikujących narzędzi analitycznych. Strona hostująca aplikację (`matiseekk-dot.github.io/Games/`) korzysta z anonimowej statystyki ruchu — patrz sekcja 4b.

4. Zewnętrzne API — RAWG.io

Aplikacja korzysta z RAWG.io API (operatora: Behind the Games, UAB, Litwa/UE) w celu wyszukiwania informacji o grach (tytuły, okładki, daty premier, gatunki). Podczas wyszukiwania zapytanie tekstowe wpisane w polu wyszukiwania trafia do serwerów RAWG. Nie są przesyłane żadne dane osobowe — jedynie tekst wyszukiwania oraz klucz API. Twoje dane (kolekcja gier) nigdy nie są przesyłane do RAWG. Polityka prywatności RAWG: rawg.io/privacy_policy.

4b. Cloudflare Web Analytics

Strona hostująca aplikację korzysta z usługi Cloudflare Web Analytics (operator: Cloudflare, Inc., USA) w celu liczenia anonimowych odwiedzin. Cloudflare Web Analytics jest zaprojektowane jako prywatność-respektujące — nie używa plików cookie, nie tworzy profili użytkowników, nie identyfikuje osób, nie śledzi między stronami. Zbierane dane: liczba wizyt, kraj (przybliżony, na podstawie IP — IP nie jest przechowywane), przeglądarka, system operacyjny, źródło ruchu (np. Google), ścieżka strony. Te dane służą wyłącznie do oceny popularności apki i nie są łączone z Twoimi danymi w aplikacji (kolekcja gier pozostaje wyłącznie lokalna). Polityka Cloudflare: cloudflare.com/privacypolicy. Możesz zablokować to śledzenie używając rozszerzeń typu uBlock Origin lub Privacy Badger — aplikacja nadal będzie działać normalnie.

5. Linki do stron trzecich

Sekcja "Wsparcie" w aplikacji zawiera link do platformy buycoffee.to/skudev umożliwiającej dobrowolne wsparcie finansowe twórcy. Kliknięcie linku otwiera stronę zewnętrzną w przeglądarce. Aplikacja nie przekazuje żadnych Twoich danych do tej platformy. Jeśli zdecydujesz się dokonać wpłaty, podlegasz polityce prywatności buycoffee.to.

6. Powiadomienia push

Jeśli włączysz powiadomienia o premierach gier, system operacyjny może wymagać przyznania odpowiedniego uprawnienia. Powiadomienia są generowane lokalnie na urządzeniu przez Service Worker — żadne dane nie są przesyłane na zewnętrzne serwery w tym celu.

7. Kontakt z twórcą (opcjonalny)

Aplikacja zawiera funkcję "Zgłoś problem" (Opcje → Informacje), która otwiera klienta poczty e-mail z gotowym szablonem zawierającym: model urządzenia (z `navigator.userAgent`), wersję aplikacji, język i liczbę gier w kolekcji. Te informacje są dołączane jedynie po to, aby pomóc w diagnozie problemów. Wysłanie wiadomości jest dobrowolne — możesz edytować lub usunąć te dane przed wysłaniem. Adres odbiorcy: skudev6@gmail.com.

8. Pliki cookie i śledzenie

Aplikacja sama w sobie nie używa plików cookie ani technologii śledzących. Statystyki Cloudflare Web Analytics (sekcja 4b) są cookieless z założenia — nie ustawiają żadnych ciasteczek ani identyfikatorów lokalnych.

9. Prawa użytkownika (RODO)

Ponieważ nie przechowujemy żadnych Twoich danych poza Twoim urządzeniem, masz pełną kontrolę nad swoimi danymi. Masz prawo do:

• dostępu — dane są widoczne w aplikacji;
• przenoszenia — przycisk "Eksportuj dane" tworzy plik JSON z całą Twoją kolekcją;
• usunięcia — przycisk "🗑️ Usuń wszystkie dane" w Opcjach → Dane (wymaga wpisania słowa USUŃ aby potwierdzić), lub odinstalowanie aplikacji.

10. Bezpieczeństwo

Dane są przechowywane w localStorage przeglądarki na Twoim urządzeniu. Bezpieczeństwo tych danych zależy od zabezpieczenia Twojego urządzenia (PIN, biometria, szyfrowanie systemu).

11. Dzieci

Aplikacja nie jest skierowana do dzieci poniżej 13 roku życia i świadomie nie zbieramy danych od takich osób. Treść aplikacji (informacje o grach z RAWG) może zawierać tytuły dla dorosłych — odpowiedzialność za dostęp dziecka spoczywa na rodzicu/opiekunie.

12. Zmiany polityki

O wszelkich istotnych zmianach w polityce prywatności poinformujemy poprzez aktualizację tej strony. Data ostatniej aktualizacji jest widoczna poniżej.

13. Kontakt

W razie pytań dotyczących prywatności skontaktuj się: skudev6@gmail.com lub przez funkcję "Zgłoś problem" w aplikacji.

PS5 Vault — Privacy Policy

✓ We don't collect your data

1. General information

PS5 Vault ("Application") is a personal game tracker for PlayStation 5. The application is free of charge and requires no registration or login. Application creator: SkuDev (Poland).

2. What data we store

All data (games list, statuses, ratings, notes, financial data, play sessions, budget) is stored locally only on your device in browser localStorage. This data is never transmitted to any external servers.

3. Data we do NOT collect

The application does not collect, store or transmit any personal data, including: name, email address, phone number, location, financial data, device identifiers, or advertising IDs. The application does not use Google Analytics, Facebook Pixel or any other identifying analytics tools. The hosting site (`matiseekk-dot.github.io/Games/`) uses anonymous traffic analytics — see section 4b.

4. External API — RAWG.io

The application uses the RAWG.io API (operator: Behind the Games, UAB, Lithuania/EU) to search for game information (titles, covers, release dates, genres). When searching, the text query you enter is sent to RAWG servers. No personal data is transmitted — only the search text and API key. Your data (game collection) is never sent to RAWG. RAWG privacy policy: rawg.io/privacy_policy.

4b. Cloudflare Web Analytics

The hosting site uses Cloudflare Web Analytics (operator: Cloudflare, Inc., USA) to count anonymous visits. Cloudflare Web Analytics is privacy-respecting by design — no cookies, no user profiles, no personal identification, no cross-site tracking. Data collected: visit count, country (approximate, based on IP — IP is not stored), browser, OS, traffic source (e.g. Google), page path. This data is used solely to assess app popularity and is never combined with your in-app data (your game collection stays strictly local). Cloudflare privacy policy: cloudflare.com/privacypolicy. You can block this tracking with extensions like uBlock Origin or Privacy Badger — the application will continue to work normally.

5. Third-party links

The "Support" section in the application contains a link to buycoffee.to/skudev, a platform allowing voluntary financial support for the creator. Clicking the link opens an external page in your browser. The application does not transmit any of your data to this platform. If you choose to make a donation, you become subject to buycoffee.to's privacy policy.

6. Push notifications

If you enable game release notifications, the operating system may require granting appropriate permission. Notifications are generated locally on the device by a Service Worker — no data is transmitted to external servers for this purpose.

7. Contact with creator (optional)

The application includes a "Report a problem" feature (Settings → Info), which opens your email client with a pre-filled template containing: device model (from `navigator.userAgent`), application version, language and number of games in the collection. This information is attached solely to help diagnose issues. Sending the message is voluntary — you can edit or remove this data before sending. Recipient address: skudev6@gmail.com.

8. Cookies and tracking

The application itself does not use cookies or any tracking technologies. Cloudflare Web Analytics statistics (section 4b) are cookieless by design — they set no cookies or local identifiers.

9. User rights (GDPR)

Since we don't store any of your data outside your device, you have full control over your data. You have the right to:

• access — data is visible in the application;
• portability — the "Export data" button creates a JSON file with your entire collection;
• deletion — the "🗑️ Delete all data" button in Settings → Data (requires typing DELETE to confirm), or by uninstalling the application.

10. Security

Data is stored in browser localStorage on your device. The security of this data depends on the security of your device (PIN, biometrics, system encryption).

11. Children

The application is not directed at children under 13 years of age and we do not knowingly collect data from such individuals. Application content (game information from RAWG) may include adult titles — responsibility for child access rests with the parent/guardian.

12. Policy changes

We will inform about any material changes to the privacy policy by updating this page. The last update date is shown below.

13. Contact

For privacy questions, please contact: skudev6@gmail.com or use the "Report a problem" feature in the application.
